▾

Answer
  • Analyse product data
  • Understand user behaviour

  • Generate insights
  • Support product decisions
Data -> Users -> Insights -> Decisions
03 How Do You Analyse A Product?
▾
Question

How do you analyse a product?

Answer
  • Start with the product objective
  • Define key metrics
  • Map the user journey
  • Identify engagement and drop-offs
  • Find insights
  • Recommend improvements
Objective -> Metrics -> Journey -> Insights
Direct Interview Version

How would you analyse the performance of a learning product?

  • Start with the goal of the product
  • Choose the right success metrics
  • Track the full learner journey
  • Check where users engage or drop off
  • Turn findings into clear recommendations

09 Learner Engagement And Completion
▾
Question

How would you analyse learner engagement and completion?

Answer
  • Track learner journey metrics
  • Measure logins, lesson views, and module completion
  • Find drop-off points
  • Compare learner groups
  • Identify friction points
  • Recommend product improvements
Direct Interview Version

How do you identify user engagement problems?

  • Look at where users stop engaging
  • Check for low clicks, views, or completion
  • Compare behaviour across different user groups
  • Find the main drop-off point
  • Use that to suggest a fix or test
10 Support A Product Launch Using Data
▾
Question

How would you support a product launch using data?

Answer
  • Define success metrics before launch
  • Track sign-ups and activation during launch
  • Monitor onboarding and early engagement
  • Analyse retention after launch
  • Recommend improvements
